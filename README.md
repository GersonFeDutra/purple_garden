# Purple Garden
Game project for the programming lab discipline. Genre: harvest sin + survival tower defense.


## Instructions

#### Requirements

* python3.8 or later
* command pip must be available

### Linux

#### Install

Simply run [./libs.sh](./libs.sh)

### Windows

Simply run [./main.bat](./main.bat)
